## Installation 
Установка через [NuGet](https://www.nuget.org/packages/SimpleSOAPClient/) package. Наберите в готовом проекте (остальное должно подтянуться)

```powershell
Install-Package SimpleSOAPClient -Pre
Install-Package ETMClient -Pre
```
